## Template

Copy this for new ADRs:

```markdown
# ADR-NNN: Title

**Status:** Proposed | Accepted | Deprecated
**Date:** YYYY-MM-DD
**Authors:** Engineer A/B

## Context
What situation forced this decision?

## Decision
What did we decide?

## Alternatives Considered
| Alternative | Why rejected |

## Consequences
What does this cost us? What does it give us?
```
